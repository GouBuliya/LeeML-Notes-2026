## 什么是Google Colab？🤔

Google Colab是一个允许你在浏览器中编写和执行Python代码的工具。它有三个主要特点：无需额外配置即可使用、提供免费的GPU资源，并且方便与他人共享协作。

## 基础操作指南 🛠️

上一节我们介绍了Colab是什么，本节中我们来看看它的基本界面和操作。

打开Colab后，你会看到由一个个“单元格”组成的界面。每个单元格可以执行不同的任务。

以下是单元格的两种主要类型及其操作：

- **代码单元格**：点击“+代码”可以生成一个新的代码单元格。在这里你可以编写和执行Python代码。
- **文本单元格**：点击“+文本”可以生成一个文本单元格。你可以在这里编写注释或对代码进行说明。

将鼠标悬停在单元格左侧时，会出现一系列工具图标，你可以使用它们来上移、下移或删除单元格。

一个代码单元格可以以两种模式运行：

1. **普通Python模式**：直接编写Python代码并执行。
2. **Shell脚本模式**：在命令前加上感叹号 `!`，可以执行Shell命令。例如，`!ls` 用于列出当前工作目录下的所有文件。

需要注意的是，以 `!` 开头的命令会启动一个新的Shell进程，执行完毕后该进程会被终止，因此每个命令都是独立的。此外，还有一个 `%` 开头的魔法命令，它会影响到整个Notebook。特别要注意，**如果要使用 `cd` 命令切换目录，必须使用 `%cd`**，这样才能永久改变工作目录。如果使用 `!cd`，则只在该条命令执行期间有效。

## 使用免费GPU 🎮

上一节我们学会了如何运行代码，本节中我们来看看如何启用Colab提供的免费GPU加速，这对于机器学习训练至关重要。

Colab提供免费的GPU资源，但默认情况下并未启用。你需要手动更改运行时设置。

以下是启用GPU的步骤：

1. 点击顶部菜单栏的“运行时”。
2. 选择“更改运行时类型”。
3. 在“硬件加速器”下拉菜单中，选择“GPU”。
4. 点击“保存”。

设置完成后，你就可以在代码中使用GPU了。GPU的型号会影响训练速度，Colab通常提供K80、T4或P100等型号。你可以运行以下代码来检查分配到的GPU型号：

```python
!nvidia-smi
```

执行后，在输出信息中找到你的GPU型号（例如，Tesla K80）。通常，T4的速度大约是K80的两倍。但请注意，使用免费版Colab时，大部分时间分配到的都是K80。

## 文件管理 📁

在机器学习项目中，我们经常需要下载作业数据或上传自己的文件。本节将介绍如何在Colab中管理文件。

Colab的运行环境是临时的。关闭浏览器或会话超时后，所有文件都会消失。因此，理解如何下载、上传文件以及连接永久存储至关重要。

### 下载文件

你可以使用 `gdown` 命令从Google Drive下载文件。你需要提供文件的ID，这个ID包含在Google Drive的分享链接中。

例如，分享链接格式通常为 `https://drive.google.com/file/d/FILE_ID/view`，其中的 `FILE_ID` 就是你需要使用的部分。

下载命令的基本格式如下，你可以指定下载后保存的文件名：

```bash
!gdown --id FILE_ID -O filename
```

### 挂载Google Drive

![](images/2e3c887ebc68.png)

![](images/4ed114c19265.png)

为了解决文件临时存储的问题，你可以将个人的Google Drive挂载到Colab环境中，这样就能直接读写云端硬盘里的文件，实现永久化存储。

![](images/30a139527751.png)

![](images/797573a4006a.png)

挂载Drive只需运行以下两行代码。实际上，你不需要手动输入，点击左侧边栏的“挂载Google云端硬盘”图标，Colab会自动为你生成这些代码。

![](images/4ab0ee7ff1aa.png)

```python
from google.colab import drive
drive.mount('/content/drive')
```

![](images/9dca2c9efa58.png)

挂载后，你的云端硬盘会出现在 `/content/drive/MyDrive` 目录下。之后所有文件操作都会同步到你的Google Drive账户中。

## 重要注意事项与限制 ⚠️

![](images/b73d0c77348d.png)

上一节我们介绍了Colab的强大功能，但“天下没有免费的午餐”，免费服务也存在一些限制。本节将讨论这些限制及其应对方法。

以下是使用免费版Colab时需要注意的关键限制和解决方案：

- **会话超时**：如果闲置时间超过90分钟（有时更短），Colab会自动断开连接。浏览器窗口关闭或屏幕休眠也可能导致断线。
  
  *解决方法*：保持屏幕常亮，或尝试运行一些自动点击页面的JavaScript代码来保持连接。
- **GPU用量限制**：免费GPU的计算资源不是无限的。如果使用量达到上限，你的账号可能会被暂时停用。
  
  *解决方法*：可以准备多个Google账号轮流使用，或者考虑升级到Colab Pro。
- **资源竞争**：免费用户分配的GPU（如K80）性能可能较弱，且需要排队。
  
  *解决方法*：升级到Colab Pro或Pro+可以获得更稳定、更强大的GPU（如P100或V100）。当然，最好的方式是尽量利用自己的本地计算资源。
