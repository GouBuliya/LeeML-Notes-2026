### 知识蒸馏

知识蒸馏的核心思想是，从一个训练良好的教师网络中学习知识。学生网络通常是一个较小的网络，我们希望通过教师网络的输出或隐藏层的表示来提取有用的信息。

最常见的方法是使用教师网络输出的概率分布作为训练目标，希望学生网络能预测出与教师网络相似的概率分布。

在本次作业中，你需要实现一个用于知识蒸馏的损失函数。我们会提供一个在 `-11` 测试集上准确率达到 89.9% 的预训练 ResNet-18 教师网络。当然，你也可以自己从头训练教师网络，但必须遵循规则：不能使用额外数据，且权重必须随机初始化。

知识蒸馏损失函数通常结合了交叉熵损失和KL散度损失，公式如下：

**总损失 = α * KL散度损失(教师输出/温度T, 学生输出/温度T) + (1 - α) * 交叉熵损失(学生输出, 真实标签)**

其中，α 和 T 是可调节的超参数。

### 架构设计：深度可分离卷积

为了构建参数更少的学生网络，我们需要在架构设计上下功夫。深度可分离卷积是一种能显著减少参数量的设计。

它分为两步：

1. **深度卷积**：在每个输入通道上独立进行空间卷积，提取局部特征，此时通道间不互通。
2. **逐点卷积**：使用 1x1 卷积来组合深度卷积输出的通道，提取通道间的信息。

通过简单的计算可以比较参数量：

- **标准卷积层参数量**：`I * O * K * K` (I: 输入通道数， O: 输出通道数， K: 卷积核大小)
- **深度可分离卷积参数量**：`I * K * K + I * O`

当卷积核 K 较大时，深度可分离卷积在参数量上具有显著优势。因此，实现深度可分离卷积是通过本次作业强基线的重要步骤。

```python
class KnowledgeDistillationLoss(nn.Module):
    def __init__(self, alpha, T):
        super().__init__()
        self.alpha = alpha
        self.T = T
        self.ce_loss = nn.CrossEntropyLoss()
        self.kl_loss = nn.KLDivLoss(reduction='batchmean')

    def forward(self, student_logits, teacher_logits, labels):
        # 计算交叉熵损失
        ce = self.ce_loss(student_logits, labels)

        # 计算KL散度损失，应用温度缩放
        student_soft = F.log_softmax(student_logits / self.T, dim=1)
        teacher_soft = F.softmax(teacher_logits / self.T, dim=1)
        kl = self.kl_loss(student_soft, teacher_soft) * (self.T ** 2)

        # 组合损失
        total_loss = (1 - self.alpha) * ce + self.alpha * kl
        return total_loss
```
