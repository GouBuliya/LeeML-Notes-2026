## 从回归到分类 🔄

![](images/b3c1c56050b4.png)

上一节我们介绍了回归模型，它接收一个向量输入并输出一个数值。我们的目标是让模型的输出值 \(\hat{y}\) 尽可能接近真实标签 \(y\)。

![](images/73c69b74649f.png)

![](images/cedd9aec4a8c.png)

![](images/9ff0f1cc1f5e.png)

一个直接的想法是将分类问题视为回归问题：为每个类别分配一个数字编号（例如，类别1=1，类别2=2，类别3=3），然后训练模型输出一个接近该编号的数值。然而，这种方法存在缺陷，因为它隐含地假设了类别之间存在顺序和距离关系（例如，类别1和2比类别1和3更“相似”），而这在许多分类问题中并不成立。

![](images/77f4b90f0553.png)

![](images/d4d816d1ced6.png)

![](images/820fc5e4f9e3.png)

## 使用独热编码表示类别 🔢

为了解决上述问题，分类问题中更常见的做法是使用**独热编码**来表示类别标签。

![](images/09ae9bdf51a0.png)

以下是独热编码的表示方法：

- 对于三个类别，标签 \(\hat{y}\) 是一个三维向量。
- 如果样本属于类别一，则 \(\hat{y} = [1, 0, 0]^T\)。
- 如果样本属于类别二，则 \(\hat{y} = [0, 1, 0]^T\)。
- 如果样本属于类别三，则 \(\hat{y} = [0, 0, 1]^T\)。

![](images/58cc8e991fb3.png)

使用独热编码后，任意两个不同类别向量之间的距离是相等的，避免了人为引入的类别关系。

![](images/379dc04b1d90.png)

![](images/6267015550be.png)

## 修改网络输出与 Softmax 函数 🧮

![](images/aee6fee8d390.png)

![](images/e8a818551a6f.png)

既然我们的目标 \(\hat{y}\) 现在是一个向量，网络也需要输出多个数值。这可以通过并行使用多组权重和偏置来实现，从而将网络输出 \(y\) 扩展为一个向量。

![](images/459ecc7cf421.png)

![](images/edeee69902f5.png)

然而，网络原始输出 \(y\) 的值域是任意的，而我们的目标 \(\hat{y}\) 的值仅在0和1之间。为了使两者可比，我们通常会在网络输出层之后添加一个 **Softmax** 函数，将输出归一化到0到1之间，并且所有输出值的和为1。这样得到的输出我们记为 \(y‘\)。

![](images/b55456231804.png)

![](images/97aeb464b396.png)

![](images/326d1930a3fa.png)

![](images/13c43c0fadb4.png)

Softmax 函数的运算过程如下：

1. 对向量的每个元素取指数：\(exp(y_i)\)，确保所有值为正。
2. 将所有指数值求和得到分母：\(sum = \sum_j exp(y_j)\)。
3. 将每个指数值除以该总和：\(y‘_i = \frac{exp(y_i)}{\sum_j exp(y_j)}\)。

用公式描述为：

\[y‘_i = \frac{e^{y_i}}{\sum_{j} e^{y_j}} 
\]

![](images/debe4d770245.png)

Softmax 不仅完成了归一化，还会放大较大值和较小值之间的差距。对于二分类问题，使用 Softmax 等价于对其中一个输出使用 Sigmoid 函数。

![](images/6a9eaab2c06c.png)

## 损失函数：MSE 与 Cross Entropy ⚖️

得到 \(y‘\) 后，我们需要计算其与真实标签 \(\hat{y}\) 之间的“距离”，即损失。常见的两种损失函数是均方误差和交叉熵。

![](images/fd5c2cb4b852.png)

**均方误差** 的计算方式与回归问题相同，即计算两个向量各维度差值的平方和：

\[L_{MSE} = \sum_i (\hat{y}_i - y‘_i)^2 
\]

![](images/83bda53883e9.png)

**交叉熵** 的计算公式则不同：

\[L_{CE} = -\sum_i \hat{y}_i \ln(y‘_i) 
\]

![](images/860b91a5b86c.png)

![](images/238dc010dbfa.png)

![](images/f1e3ce0cd161.png)

![](images/90fbbd4fea62.png)

![](images/40918a574ab0.png)

当 \(y‘\) 与 \(\hat{y}\) 完全相同时，两种损失都能达到最小值。但在分类任务中，交叉熵被更广泛地使用。从优化角度看，交叉熵能提供更有效的梯度信号。

![](images/8defab42f4ea.png)

![](images/29b016c2fba6.png)

![](images/329edde52a31.png)

![](images/f38177a16c79.png)

## 为什么 Cross Entropy 更优？ 📈

![](https://github.com/OpenDocCN/dsai-notes-zh/raw/master/docs/leemldl-2026/img/de1f8b64fdaa484f64f94e79320d49e0_73.png)

![](images/9b5c9b8c28d7.png)

![](images/bc721eb356d1.png)

![](images/7cc5fc8c8e9b.png)

![](images/2814c1e8c5a3.png)

我们可以通过一个简单的例子来直观理解。考虑一个三分类问题，固定 \(y_3 = -1000\)，观察 \(y_1\) 和 \(y_2\) 变化时，两种损失函数构成的误差曲面。

![](images/a7e1443cb8c5.png)

![](images/b759b8635f76.png)

![](images/75f3f05cb0a0.png)

![](images/4a0fcf9c3bbb.png)

以下是两种损失函数误差曲面的对比：

- 当使用 **交叉熵损失** 时，即使在损失值很大的区域（对应预测错误），误差曲面也有明显的斜率，梯度下降法可以顺利地向损失小的区域移动。
- 当使用 **均方误差** 时，在损失值很大的区域，误差曲面可能非常平坦，梯度接近于零。这会导致梯度下降法更新缓慢甚至停滞，使得训练难以启动或收敛速度很慢。

![](images/8ec36f20bb9d.png)

![](images/aeea2d4f6239.png)

![](images/54cc7937ccc1.png)

![](images/b55e6cf097a0.png)

因此，在分类问题中，使用交叉熵损失通常能使模型训练得更顺利、更快速。在实践中，交叉熵损失与 Softmax 函数经常被绑定使用。例如在 PyTorch 中，调用交叉熵损失函数时会自动在网络最后一层加上 Softmax 运算。

本节课中我们一起学习了如何将神经网络应用于分类问题。关键步骤包括使用独热编码表示类别标签，在网络末端添加 Softmax 函数进行归一化，以及选择交叉熵作为损失函数以获得更好的训练动态。理解这些组件如何协同工作，是构建有效分类模型的基础。
