## 作业目标 🎯

本次作业的主要目标有以下几点：

- 使用深度神经网络架构解决一个回归问题。
- 了解并实践机器学习训练过程中的各种技巧，例如参数调整和特征选择。
- 学习并掌握PyTorch框架的基本使用方法。

## 数据集介绍 📊

我们使用的数据来自CMU团队通过Facebook问卷收集的COVID-19相关数据。该数据集包含了美国各州每日的多种特征，例如类似COVID的症状比例、社交距离行为、心理健康指标等。

以下是数据集中的一些特征示例：

- `covid_like_illness`: 出现类似COVID症状的比例。
- `behavior_wear_mask`: 佩戴口罩的行为比例。
- `travel_outside_state`: 州外旅行的比例。
- `mental_health_indicator`: 心理健康指标（如紧张、抑郁情绪）。
- `tested_positive`: 确诊阳性比例（这是我们预测的目标标签）。

数据中的“州”信息使用独热编码表示。例如，亚利桑那州可能被编码为 `[0, 0, 1, 0, ...]`。

**重要提示**：请勿从原始来源下载数据用于训练。作业所需的数据已由助教处理并提供。

## 评分标准 📈

模型性能的评估采用均方误差。其计算公式如下：

```
MSE = (1/n) * Σ(y_pred - y_true)²
```

![](images/b122e835f603.png)

![](images/bfd3a2a011dd.png)

其中，`y_pred` 是模型的预测值，`y_true` 是真实值。该值越小，代表模型预测越准确。

## 作业评分基准线

本次作业没有书面报告，分数分布在以下几个基准线上：

- **Simple Baseline**: 运行示例代码即可通过。
- **Medium Baseline**: 需要实施有效的特征选择。
- **Strong Baseline**: 需要调整模型架构或优化器等超参数。
- **Boss Baseline**: 需要实现正则化等更高级的技巧。
- **代码提交**: 将最终代码提交到课程网站NTU COOL。

### 1. 工具函数定义

首先定义了一些辅助函数：

- `set_seed`: 固定随机种子，确保实验结果可复现。
- `train_valid_split`: 从训练集中划分出验证集，用于评估模型泛化能力。
- `predict`: 将模型的预测结果保存为提交所需的CSV文件。

### 4. 特征选择

`select_feature` 函数用于选择用于训练的特征。当参数为 `False` 时，可以手动指定特征索引列表。实现有效的特征选择是通过Medium Baseline的关键。

### 5. 训练循环

以下是训练过程的核心步骤：

- **定义损失函数与优化器**：使用均方误差损失和随机梯度下降优化器。优化器的 `weight_decay` 参数可用于实现L2正则化（Boss Baseline要求）。
- **训练模式与评估模式**：使用 `model.train()` 和 `model.eval()` 进行切换。在评估模式下，模型会关闭Dropout等仅在训练时使用的层。
- **早停机制**：当验证集损失在指定步数内不再下降时，提前终止训练以防止过拟合。

### 6. 超参数配置

所有可调整的超参数集中在 `config` 字典中，方便进行实验，包括：

- `seed`: 随机种子。
- `select_all`: 是否使用全部特征。
- `n_epochs`: 训练轮数。
- `batch_size`: 批大小。
- `learning_rate`: 学习率。
