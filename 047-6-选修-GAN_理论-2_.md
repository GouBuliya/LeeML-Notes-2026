# 47：6-选修-GAN 理论-2 🧠

![](images/e9244600c01e.png)

在本节课中，我们将学习生成对抗网络（GAN）理论的一个重要扩展——f-GAN。我们将探讨如何用更广泛的f散度（f-divergence）来衡量生成分布与真实分布之间的差异，并理解其背后的数学原理。虽然这部分内容数学性较强，但它能帮助我们更深入地理解GAN的优化目标。

## 概述 📋

![](images/f2b370a65fd1.png)

上一节我们介绍了GAN的基本框架和JS散度。本节中，我们来看看如何将GAN的框架推广到使用任意的f散度。f-GAN的核心思想是：我们可以用任何满足特定条件的f散度来替代JS散度，作为衡量生成数据与真实数据分布差异的指标。

![](images/346bd84f3df5.png)

![](images/1e5d595568b3.png)

![](images/fb7287132705.png)

## 什么是f散度？ 🤔

![](images/5854b4a1cef8.png)

![](images/0878f08c4047.png)

![](images/622ed3efe24e.png)

f散度是衡量两个概率分布P和Q之间差异的一种通用方法。给定一个凸函数f，且满足f(1)=0，两个分布P和Q之间的f散度定义为：

![](images/d7db6fda190a.png)

![](images/7de14398d15d.png)

![](images/8044a8fa6623.png)

![](images/f4989df0e61d.png)

**公式：**  

`D_f(P || Q) = ∫_x q(x) * f( p(x) / q(x) ) dx`

![](images/837152c3cb5e.png)

![](https://github.com/OpenDocCN/dsai-notes-zh/raw/master/docs/leemldl-2026/img/bddadfa40708e93e33c7031e0592282d_26.png)

其中，`p(x)`和`q(x)`分别是分布P和Q在点x的概率密度。

![](images/84c62c94c5be.png)

这个定义满足作为“距离”的基本性质：当P和Q完全相同时，散度值为0；当它们不同时，散度值大于0。

## 共轭函数（Conjugate Function） 🔄

![](images/25c0e072ea51.png)

为了将f散度与GAN联系起来，我们需要引入一个重要的数学概念——共轭函数。对于每一个凸函数f，都存在一个对应的共轭函数f*。

![](images/a94be2d40407.png)

![](images/ab395ac9557f.png)

![](images/fc3d4274c2b0.png)

共轭函数f*的定义如下：

![](images/be7dc7e5603e.png)

![](https://github.com/OpenDocCN/dsai-notes-zh/raw/master/docs/leemldl-2026/img/bddadfa40708e93e33c7031e0592282d_40.png)

![](images/16fac8e9abaf.png)

![](images/085ef3ffac0c.png)

![](images/f700124db6fd.png)

**公式：**  

`f*(t) = max_{x∈dom_f} { t*x - f(x) }`

![](images/dcba9c293f3b.png)

![](images/962458de9853.png)

![](images/5569324134d4.png)

直观上，对于给定的t，我们寻找一个x，使得`t*x - f(x)`的值最大。这个最大值就是f*(t)。共轭函数f*本身也是一个凸函数。

![](images/c37b6bffa916.png)

![](images/9bad6a88744e.png)

例如，如果`f(x) = x * log x`，那么它的共轭函数是`f*(t) = exp(t-1)`。

![](images/337ecb1660e0.png)

![](images/bfafb8c40fc8.png)

![](https://github.com/OpenDocCN/dsai-notes-zh/raw/master/docs/leemldl-2026/img/bddadfa40708e93e33c7031e0592282d_62.png)

## f散度与GAN的联系 ⛓️

讲完f散度和共轭函数，它们似乎与GAN毫无关系。接下来我们就要进入与GAN有关的核心内容了。

![](images/4efcf7329f51.png)

![](images/e995f6fd256c.png)

关键在于，我们可以利用共轭函数，将f散度的定义进行变换。经过一系列推导（利用共轭函数定义替换f），原始的f散度可以转化为以下形式：

![](images/b634496e46f4.png)

**公式：**  

`D_f(P_data || P_g) = max_D { E_{x~P_data}[ D(x) ] - E_{x~P_g}[ f*( D(x) ) ] }`

![](images/358cb8fb0525.png)

![](images/86993889c553.png)

![](images/189599d98fd3.png)

这个形式是不是看起来非常眼熟？它与原始GAN中判别器D要最大化的目标函数形式一致！

![](images/3c5b0345c53d.png)

- `E_{x~P_data}[ D(x) ]` 表示判别器对真实数据的打分期望。
- `E_{x~P_g}[ f*( D(x) ) ]` 是一个与共轭函数f*相关的项，作用于生成数据。

![](images/9ce97c70eb27.png)

**核心结论：**  

在GAN的训练中，判别器D试图最大化上述目标函数，以逼近真实的f散度 `D_f(P_data || P_g)`。而生成器G则试图最小化这个被逼近的f散度。**选择不同的凸函数f（即不同的f散度），就对应着选择不同的共轭函数f*，从而定义了GAN训练中不同的目标函数V(D, G)。**

![](images/eea67c6b997e.png)

以下是几种常见散度及其对应的f和f*函数：

![](images/6ab72df14cbc.png)

![](images/026fe1bf4310.png)

- **KL散度**： `f(x) = x log x`， `f*(t) = exp(t-1)`
- **Reverse KL散度**： `f(x) = -log x`， `f*(t) = -log(1-t)`
- **JS散度**： `f(x) = x log x - (x+1)log((x+1)/2)`， `f*(t) = -log(2 - exp(t))`

![](images/3e14db8f4a93.png)

## 模式崩溃（Mode Collapse）的探讨 💥

![](images/7ebf54aaa06a.png)

f-GAN的理论为我们提供了一个新视角来理解GAN训练中的经典难题——模式崩溃。

![](images/1ca6fd2bd3f9.png)

![](images/ed3fe69387d3.png)

- **模式坍塌（Mode Collapse）**： 生成器只生成非常相似或单一的样本，多样性严重不足。
- **模式丢弃（Mode Dropping）**： 真实数据分布有多个簇（例如不同肤色的人脸），但生成器只能覆盖其中一部分簇。

![](images/ad69b6833ca4.png)

![](https://github.com/OpenDocCN/dsai-notes-zh/raw/master/docs/leemldl-2026/img/bddadfa40708e93e33c7031e0592282d_95.png)

一种观点认为，模式崩溃可能与选择的散度有关。例如：

- 最小化 **KL散度** 可能导致生成分布“覆盖”所有真实数据点，甚至在低概率区域也产生样本，这可能对应生成模糊的图片。
- 最小化 **Reverse KL散度** 则倾向于让生成分布紧密贴合真实分布的某一个高峰（模式），而忽略其他模式。这恰好类似于模式崩溃的行为。

![](images/3679ac5064af.png)

![](images/60f6705a5f76.png)

![](https://github.com/OpenDocCN/dsai-notes-zh/raw/master/docs/leemldl-2026/img/bddadfa40708e93e33c7031e0592282d_100.png)

![](images/441c73a4f046.png)

传统的GAN使用的JS散度，其性质更接近Reverse KL散度，这或许部分解释了为什么GAN容易发生模式崩溃。

![](images/64c509d4ca49.png)

![](images/61bca3336fc8.png)

**那么，f-GAN能解决模式崩溃吗？**  

理论上，既然我们可以自由选择散度（比如换成KL散度），似乎应该能缓解问题。但实践表明，简单地更换散度并不能根本解决模式崩溃问题，这说明模式崩溃的成因更为复杂。

![](images/e77925cd7b72.png)

## 实践建议：一种应对模式崩溃的技巧 🛠️

![](https://github.com/OpenDocCN/dsai-notes-zh/raw/master/docs/leemldl-2026/img/bddadfa40708e93e33c7031e0592282d_110.png)

![](images/c3f2742b28d2.png)

在做相关作业时，你很可能遇到模式崩溃问题。这里提供一个在实践中简单有效的技巧：**训练多个生成器（Ensemble）**。

![](images/dddc39cd023a.png)

以下是具体操作思路：

1. 独立地训练多个GAN模型（例如25个），每个模型都有自己的生成器。
2. 由于模式崩溃的随机性，每个生成器可能会塌缩到不同的模式（例如生成不同风格的人脸）。
3. 当需要生成多样化的样本时，从这多个生成器中各取一个样本进行组合。
4. 对于用户而言，他们看到的是多样化的输出，而并不知道背后有多个生成器。

![](images/c8123bf4d62a.png)

![](images/d971e3f0ad7e.png)

虽然这需要训练更多模型，增加了计算成本，但它是确保输出多样性的一种可靠方法。

![](images/018e18b3b64c.png)

![](images/a2ce88fac216.png)

![](images/44708cc86f24.png)

## 总结 🎯

![](images/1d468e926488.png)

![](images/ff0cc6b1b650.png)

![](images/068fb7e989d3.png)

![](images/ee1d562fbe70.png)

本节课我们一起学习了f-GAN的理论。我们从f散度的定义出发，引入了共轭函数的概念，并推导出f散度可以转化为一个与GAN目标函数相似的最大化问题。这揭示了GAN的对抗训练过程本质上是在优化一个特定的f散度。不同的f散度选择对应着不同的优化目标。我们还探讨了模式崩溃问题与散度选择之间的潜在联系，并介绍了一种实用的集成方法来缓解模式崩溃。尽管f-GAN在理论上非常优美，但实际应用中更换散度带来的改进有限，模式崩溃的彻底解决仍需更多研究。
