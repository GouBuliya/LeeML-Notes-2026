# 3：深度学习基本概念（下） 🧠

![](images/e90988a7cdae.png)

在本节课中，我们将要学习如何突破线性模型的限制，构建更复杂、更有弹性的模型。我们将从线性模型的不足出发，逐步引入分段线性函数、Sigmoid函数以及神经网络的基本概念，最终理解深度学习模型是如何构建和训练的。

## 线性模型的限制与模型偏差 🚫

![](images/5b44d2376d8b.png)

上一节我们介绍了线性模型的基本形式。线性模型可能过于简单。我们可以想象，特征X1和输出Y之间可能存在比较复杂的关系。但对线性模型而言，X1和Y的关系就是一条直线。随着X1增加，Y就线性增加。你可以通过调整权重W改变直线的斜率，通过调整偏置B改变直线与Y轴的交点。但无论你怎么调整W和B，它永远都是一条直线。这意味着X1越大，Y就越大。例如，前一天观看人数越多，隔天观看人数就越多。

![](images/194907284651.png)

但现实可能并非如此。也许在X1小于某个阈值时，前一天和隔天的观看人数成正比。但当X1超过某个峰值后，观看人数反而可能减少。也就是说，X1和Y之间可能存在一个像红色曲线那样复杂的关系。然而，无论你如何调整线性模型的参数，你永远无法用线性模型制造出那条红色曲线。

![](images/e63f462a0fcf.png)

所以，线性模型有很大的限制。这种来自于模型本身的限制，叫做**模型的偏差**。请注意，这里的“模型偏差”与线性方程中的参数“偏置B”是两个不同的概念，虽然用词相同但含义不同。模型的偏差指的是模型本身无法模拟真实复杂关系的能力缺陷。

## 构建更复杂的函数：从分段线性到Sigmoid 🧩

既然线性模型不够好，我们需要写出一个更复杂、更有弹性的、带有未知参数的函数。

![](images/ca0f4ec41925.png)

我们可以观察那条红色曲线。它可以看作是一个常数项，再加上一系列蓝色的函数组合而成。这个蓝色函数的特性是：当输入值小于某个阈值时，输出是一个定值；大于另一个阈值时，输出是另一个定值；中间有一个斜坡区域。我们暂时称它为“蓝方”。

![](images/28a608611f34.png)

![](images/f0a21e0cb846.png)

![](images/1b7bf1fefe3c.png)

![](images/522df77f9ee9.png)

以下是构建红色曲线的步骤：

1. 常数项的值设为红色曲线与X轴的交点。
2. 第一个蓝色函数的斜坡起点设在红色曲线段的起点，斜坡终点设在第一个转折点，并使其斜率与红色线段一致。将常数项与第一个蓝色函数相加，就得到了第一个转折点之前的红色线段。
3. 第二个蓝色函数的斜坡设在第一个转折点到第二个转折点之间，斜率与对应红色线段一致。将常数项、第一个和第二个蓝色函数相加，就得到了两个转折点之间的红色线段。
4. 第三个蓝色函数的斜坡起点设在第二个转折点，斜率与之后红色线段的斜率一致。将所有项相加，就得到了完整的红色曲线。

![](images/e75f171685bc.png)

![](images/7b5197032532.png)

![](images/6f1084ed5a82.png)

![](https://github.com/OpenDocCN/dsai-notes-zh/raw/master/docs/leemldl-2026/img/90263d1cea57d2d326f4492bfd48cd8f_25.png)

![](images/09c9b0fec127.png)

![](images/d0f27d58d8a0.png)

所以，红色曲线可以看作是一个常数项加上一堆蓝色函数的组合。

![](images/ae109d137f71.png)

![](images/25cd09e057aa.png)

![](images/6724e1a5aa52.png)

![](images/523c69411e68.png)

进一步思考，你会发现，任何**分段线性曲线**都可以用常数项加一大堆不同的蓝色函数组合出来。分段线性曲线越复杂（转折点越多），所需的蓝色函数就越多。

有人可能会说，我们考虑的关系可能不是分段线性的，而是连续的平滑曲线。这也没关系，我们可以在曲线上取足够多的点，用分段线性曲线去逼近它。点取得越多、位置越恰当，逼近就越精确。

![](images/ccf151469cf6.png)

因此，我们得出一个重要结论：**可以用分段线性曲线去逼近任何连续曲线**。而每一个分段线性曲线，又都可以用一大堆蓝色函数组合起来。也就是说，只要有足够的蓝色函数相加，我们就可以组合出任何连续的曲线。

## Sigmoid函数：蓝色函数的近似表达 📈

![](images/9871eef850e6.png)

![](images/6d968f7d4afa.png)

![](images/4c46fd0a678e.png)

接下来的问题是，如何用数学式子写出这个蓝色函数？直接写出它的表达式可能不那么容易，但我们可以用一条曲线来逼近它。

![](images/9ced089b95c2.png)

![](images/942ef4055588.png)

我们使用一个**Sigmoid函数**来逼近这个蓝色函数。Sigmoid函数的表达式如下：

![](images/6587e9d3cba2.png)

![](images/3a5fa9ab5cad.png)

**公式：**  

`y = c * sigmoid(b + w * x1)`  

其中，`sigmoid(z) = 1 / (1 + exp(-z))`

![](images/5e163c3199da.png)

![](images/258a59c9f88f.png)

![](images/caa0febb99a3.png)

所以完整形式是：  

`y = c / (1 + exp(-(b + w * x1)))`

![](images/7166b30e5d30.png)

![](images/e8cc38b31520.png)

![](images/177cc9d5ad30.png)

![](images/79dc5822e553.png)

![](images/718ffe3eac3d.png)

![](images/50b6245eacaa.png)

- 当输入x1趋近于正无穷大时，输出y会收敛到高度c。
- 当输入x1趋近于负无穷大时，输出y会趋近于0。

![](images/b38c909c9b88.png)

![](images/0461d259cdd8.png)

这个函数形状像字母“S”，因此得名Sigmoid（S型函数）。而我们要逼近的那个蓝色函数，通常被称为**Hard Sigmoid**。

![](images/73985786c58f.png)

我们可以通过调整Sigmoid函数中的参数来制造不同形状的曲线：

- 改变**w**：会改变斜坡的坡度。
- 改变**b**：会将函数曲线左右移动。
- 改变**c**：会改变函数的高度。

![](images/593cc7d81fd7.png)

通过组合不同参数（w, b, c）的Sigmoid函数，我们就可以组合出各种不同的分段线性函数，进而逼近各种不同的连续函数。

## 神经网络的数学表示与结构 🕸️

![](images/dcfb7eccafa9.png)

现在，我们可以将红色曲线的函数正式写出来。它由常数项和多个蓝色函数（用Sigmoid近似）相加而成。

![](images/14b6c523e5d9.png)

![](images/a609e0ab05db.png)

**公式：**  

`y = b + Σ_i [ c_i * sigmoid(b_i + w_i * x1) ]`  

其中，`i` 代表第i个Sigmoid函数，每个函数都有自己的参数 `c_i`, `b_i`, `w_i`。

当我们有多个输入特征（例如，考虑前j天的数据）时，公式可以扩展为：

![](images/90b3347415e1.png)

**公式：**  

`y = b + Σ_i [ c_i * sigmoid(b_i + Σ_j w_{ij} * x_j) ]`  

这里，`j` 代表特征编号，`i` 代表第i个Sigmoid函数。`w_{ij}` 是第i个函数中，乘给第j个特征的权重。

![](images/c09973c64627.png)

这个公式可以用更直观的**网络图**来表示，也可以用**线性代数**简洁地表示。

![](images/b85a87ce89b5.png)

![](images/d5fd8a5efda4.png)

**线性代数表示（前向传播过程）：**

1. `r = W * x + b` （计算净输入）
2. `a = sigmoid(r)` （通过激活函数）
3. `y = c^T * a + b` （加权求和得到输出）

其中，`x` 是输入特征向量，`W` 是权重矩阵，`b` 是偏置向量，`r` 是中间结果，`a` 是激活值，`c` 是输出层权重向量，最后的 `b` 是输出层偏置（标量）。

![](images/b33c091f54e6.png)

我们将所有权重和偏置参数（W, b, c, b）拉直，组合成一个很长的向量，统称为参数 `θ`。机器学习的第一步——定义带有未知参数的函数，就此完成。

![](images/2ac1dfe0f2df.png)

![](images/0daaa360d28f.png)

## 模型的训练：损失函数与梯度下降 ⚙️

![](images/7ee4fe28418d.png)

上一节我们定义了新的模型，接下来进入第二步和第三步：定义损失函数和进行优化。

![](images/d44f72a5d8e0.png)

**定义损失函数：**  

损失函数的定义方法与之前线性模型时完全相同，只是参数变成了 `θ`。我们计算模型预测值 `y` 与真实标签之间的差距（如均方误差），并对所有训练数据求和，得到总损失 `L(θ)`。

![](images/b670a6e044be.png)

**优化（梯度下降）：**  

优化目标仍然是找到一组参数 `θ*`，使得损失 `L(θ)` 最小。我们依然使用**梯度下降法**。

1. 随机初始化参数 `θ^0`。
2. 计算损失函数 `L` 对每个参数 `θ_i` 的偏导数，这些导数集合起来称为**梯度（gradient）**，记作 `g`。
3. 更新参数：`θ^1 = θ^0 - η * g`。其中 `η` 是学习率。
4. 重复步骤2和3，直到收敛或达到停止条件。

![](images/e27b3af48a38.png)

![](images/7ffc22b24a90.png)

![](images/c29dcb467c3a.png)

![](images/25adbdaee27d.png)

在实际操作中，我们通常不会使用全部数据计算梯度，而是采用**小批量梯度下降**。

- 将全部N笔数据随机分成若干**批次**，每个批次包含B笔数据。
- 每次迭代，我们只用一个批次的数据计算损失 `L^1` 和梯度，并用这个梯度更新参数。
- 遍历所有批次一次，称为一个**周期**。
- 每次参数更新称为一次**迭代**。

![](images/b5e24b8e9665.png)

![](images/1c29e218b6c5.png)

批次大小 `B` 和学习率 `η` 一样，是需要我们手动设定的**超参数**。

![](images/2796effaa287.png)

## 激活函数：从Sigmoid到ReLU 🔄

![](images/8b05636852a2.png)

![](images/b79d9cbaf1c4.png)

![](images/47cf6fe6cc44.png)

之前有同学问到，为什么一定要用Sigmoid来逼近Hard Sigmoid？其实不一定，我们有其他选择。

![](images/01e74dcee140.png)

![](images/9c7b6a48db7a.png)

![](images/c32e86a92430.png)

![](images/730170d8575e.png)

Hard Sigmoid 可以看作是两个 **ReLU函数** 的组合。ReLU函数的表达式为：  

**公式：** `y = c * max(0, b + w * x1)`

![](images/8182b89a744f.png)

![](images/5b3d564ff734.png)

![](images/0dfe7727a289.png)

![](images/0953db1198a1.png)

![](images/8b84d9ba8774.png)

它的形状是一条水平线，在某个点转折后变为斜坡。调整参数可以改变其位置和斜率。将两个ReLU函数叠加，就可以得到一个Hard Sigmoid函数。

![](images/5082f9a49e33.png)

![](images/a20614ae4180.png)

![](images/073ce30dc8e2.png)

![](images/399f1fa2853a.png)

因此，我们完全可以在模型中使用ReLU代替Sigmoid。Sigmoid和ReLU在机器学习中统称为**激活函数**。在后续实验中，我们选择了ReLU，因为它通常表现更好（原因后续课程会讲）。

![](images/2e408180c8ee.png)

![](images/8e46c453b264.png)

![](images/438b50cbc081.png)

## 深度神经网络与过拟合 🏗️

![](images/eb0b512d6cf2.png)

![](images/3d7fbea222f6.png)

![](images/d519a06fffd1.png)

![](images/3cb52f097465.png)

我们可以对模型做进一步的变形：将同样的运算（乘权重、加偏置、通过激活函数）反复多次。  

例如：`x -> (W1, b1, ReLU) -> a -> (W2, b2, ReLU) -> a’ -> ... -> y`  

这样我们就得到了多个**隐藏层**。这种多层的结构就是**深度神经网络**，“深度”一词便来源于此。

![](images/261c5494b09b.png)

![](images/f0e8d693bb7b.png)

![](images/b85ebd55b8f7.png)

实验表明，增加层数（使用更深网络）可以在训练数据上获得更低的损失。例如，从1层到3层，训练损失显著下降。

![](images/506378c7e326.png)

但是，当层数增加到4层时，出现了新情况：在训练数据上，4层比3层好；但在没见过的测试数据上，4层反而比3层差。这种模型在训练集上表现很好，在测试集上表现较差的现象，称为**过拟合**。这是机器学习中需要解决的核心问题之一。

![](images/b135cd165025.png)

![](images/fd3a5deccd86.png)

![](images/28b818b2bf12.png)
